# admin后台管理的运行

## 项目运行

- 安装环境：npm install
- 运行： npm run start
- 运行json-server
    - 进入code文件：cd code
    - 运行：json-server --watch .\db.json --port 8000

## 项目开发

环境配置：

- npm i --save sass

- npm i --save http-proxy-middleware

- npm i --save react-router-dom

- npm i --save antd

安装json-server

- npm install -g json-server
